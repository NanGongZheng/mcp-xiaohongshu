# 即刻推广文案

## 文案一：技术向

做了一个小红书 MCP Server，让 Claude 能直接搜小红书。

接入后可以这样用：
• "帮我搜小红书上关于护肤的热门笔记"
• "看看这个用户的主页信息"
• "现在有什么热门话题"

技术上用了可插拔数据源架构，一行命令切换 Apify 真实数据，手写 JSON-RPC 2.0 零依赖。

开源地址：https://github.com/NanGongZheng/mcp-xiaohongshu

## 文案二：场景向

做运营/营销的同学看过来 👀

我做了一个 MCP Server，让 AI 能直接查小红书数据。接入 Claude 后，用自然语言就能：
• 搜索特定关键词的热门笔记
• 查看 KOL 的粉丝数、获赞数、笔记数
• 获取当前热门话题

不用写代码，不用开后台，直接和 AI 对话就行。

开源项目，5 分钟跑通：https://github.com/NanGongZheng/mcp-xiaohongshu

## 文案三：创业思考向

AI 时代的流量入口正在迁移。

PC 时代靠搜索（SEO），移动时代靠推荐（抖音算法），AI 时代靠大模型引用。谁的 Server 被大模型调用，谁就掌握了新的流量入口。

我正在做 MCP Server 创业，第一个产品是小红书数据 Server——让 AI 能搜索和分析小红书内容。

目前框架已开源，后续会接入真实数据源，做内容分析、竞品监控、KOL 筛选等高价值场景。

有兴趣的可以关注：https://github.com/NanGongZheng/mcp-xiaohongshu
